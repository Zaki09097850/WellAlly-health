## 功能概述

中医体质模块提供体质辨识、特征分析和养生建议。

### 核心功能

1. **体质辨识** - 9种体质类型（平和、气虚、阳虚等）
2. **体质特征** - 常见表现、心理特征、发病倾向
3. **养生建议** - 饮食、起居、运动、情志
4. **中药调理** - 推荐方剂、常用中药、穴位保健

---

## 9种体质类型

1. **平和质** - 阴阳气血调和
2. **气虚质** - 元气不足、疲乏无力
3. **阳虚质** - 阳气不足、畏寒怕冷
4. **阴虚质** - 阴液亏虚、口燥咽干
5. **痰湿质** - 痰湿凝聚、体型肥胖
6. **湿热质** - 湿热内蕴、面油痤疮
7. **血瘀质** - 血行不畅、面色晦暗
8. **气郁质** - 气机郁滞、情绪低落
9. **特禀质** - 先天失常、易过敏

## 命令接口

```bash
/tcm assess                               # 进行体质辨识
/tcm diet                                 # 查看饮食建议
/tcm exercise                             # 查看运动建议
/tcm acupoints                            # 查看穴位保健
/tcm status                               # 查看中医体质状态
/tcm trend                                # 查看体质变化趋势
/tcm herbal                               # 查看中药调理建议
/tcm recommendations                      # 获取综合养生建议
```

## 功能特点

✅ **完整的9种体质类型定义**
- 基于《中医体质分类与判定》国家标准
- 每种体质包含形体特征、心理特征、发病倾向、适应能力
- 60题标准化问卷,准确评估

✅ **全面的养生建议**
- 饮食调养: 宜食/忌食食物,推荐食谱
- 起居调摄: 作息、环境、生活习惯
- 运动锻炼: 推荐运动、频次、强度
- 情志调摄: 情绪管理、心理调节
- 穴位保健: 推荐穴位、按摩方法、艾灸建议
- 中药调理: 推荐方剂、方剂组成、用法用量、注意事项

✅ **强大的分析功能**
- 体质变化趋势追踪
- 兼夹体质识别
- 调理效果评估
- 与其他健康指标关联分析

✅ **严格的医学安全**
- 完整的医学免责声明
- 明确的能力范围界定
- 中药安全警告
- 就医引导建议
