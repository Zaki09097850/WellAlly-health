## 常见问题

### Q: 这个项目是 Anthropic 官方的吗？
**A:** 不是。这是一个独立的开源项目，与 Anthropic 或 Claude.ai 没有任何关联。

### Q: 我的数据会上传到云端吗？
**A:** 不会。所有数据都存储在你的本地文件系统中。

### Q: 需要 Python/Node.js 环境吗？
**A:** 不需要。只需要安装 Claude Code CLI。

### Q: 可以用于商业用途吗？
**A:** 可以，项目使用 MIT 许可证。

## 社区准则

1. **尊重他人** - 保持友善和建设性
2. **相关话题** - 保持讨论与项目相关
3. **搜索先行** - 发布前先搜索是否已有类似讨论
4. **隐私意识** - 不要在讨论中分享真实医疗数据
